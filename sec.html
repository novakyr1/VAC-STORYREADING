1 Mark Questions

1 Mark What is the difference between "With Previous" and "After Previous" in animation timing?
"With Previous" starts the animation at the same time as the previous animation, while "After Previous" starts the animation after the previous animation has completed.
1 Mark What is the purpose of the "Selection Pane" in PowerPoint?
The Selection Pane allows you to view and manage all objects on a slide, making it easier to select, rename, show, or hide objects, especially when they are layered or overlapping.
1 Mark What is the difference between a theme and a template in PowerPoint?
A theme is a set of design elements (colors, fonts, effects) that can be applied to a presentation. A template is a pre-designed presentation file (.potx) that contains themes, sample content, and layouts that can be used as a starting point for new presentations.
1 Mark What is the benefit of using the "Format Painter" tool?
The Format Painter allows you to quickly copy formatting from one object and apply it to another, saving time and ensuring consistency in your presentation's appearance.
1 Mark What is the purpose of the "Align" tool in PowerPoint?
The Align tool helps you precisely position objects on a slide by aligning them to each other or to the slide itself (left, center, right, top, middle, bottom, or distribute evenly).
1 Mark What is the purpose of the "Hide Slide" feature and how is it used?
Hide Slide allows you to exclude specific slides from the presentation without deleting them. Right-click a slide in the thumbnail pane and select "Hide Slide" to toggle this setting.
1 Mark What is the "Presenter View" and what information does it display?
Presenter View is a special presentation mode that shows the current slide, next slide, speaker notes, and presentation timer to the presenter, while the audience sees only the current slide.
1 Mark What is the "Morph" transition and what are its requirements?
Morph is a transition that creates smooth animation between slides by morphing similar objects. It requires PowerPoint 2019/365 and objects with the same name on consecutive slides.
1 Mark What is the difference between "Embed" and "Link" when inserting media files?
Embedding stores the media file within the presentation (increasing file size), while linking creates a reference to an external file (keeping presentation size smaller but requiring the link to remain intact).
1 Mark What is the purpose of the "Format Background" options?
Format Background allows you to customize slide backgrounds with solid colors, gradients, pictures, or textures, and apply artistic effects to create visually appealing slides.

5 Mark Questions

5 Marks How can you create a custom motion path for an object?
Select the object, go to Animations tab, click "Add Animation", choose "Motion Paths", and then select "Custom Path". You can then draw your desired path on the slide.
5 Marks How do you create a hyperlink within a presentation?
Select the text or object you want to hyperlink, right-click and choose "Hyperlink", then select "Place in This Document" and choose the specific slide you want to link to.
5 Marks How can you add a video to your presentation and control its playback?
Use Insert > Video to add a video. Then use the Video Tools Playback tab to set options like Start (On Click or Automatically), trimming, fade in/out, and volume controls.
5 Marks How can you add speaker notes to your presentation?
Click in the notes pane below each slide in Normal view, or go to View > Notes Page for a larger view where you can add and format speaker notes.
5 Marks What is the purpose of the "Animation Painter" and how is it used?
The Animation Painter copies animation effects from one object to another. Select the object with the desired animation, click Animation Painter in the Animations tab, then click the target object to apply the same animation.
5 Marks How can you create a looping animation that repeats continuously?
Select the animated object, go to the Animations tab, click Animation Pane, right-click the animation effect, choose Effect Options, go to the Timing tab, and set "Repeat" to "Until End of Slide" or a specific number of times.
5 Marks How do you create a custom theme in PowerPoint?
Go to Design tab, expand the Themes gallery, and select "Save Current Theme". This saves your custom colors, fonts, and effects as a reusable theme file.
5 Marks How can you add a footer with slide numbers to all slides?
Go to Insert > Header & Footer, check the "Slide number" and "Footer" boxes, enter footer text if desired, and click "Apply to All" to add them to every slide.
5 Marks How can you rehearse and set slide timings for an automatic presentation?
Go to Slide Show > Rehearse Timings to practice your presentation. PowerPoint will record the time spent on each slide, which can be used for automatic slide advancement.
5 Marks How do you create a custom chart template in PowerPoint?
Create and format a chart as desired, right-click the chart, select "Save as Template", give it a name, and it will be available in the Chart Type dialog for future use.

10 Mark Questions

10 Marks How can you create a custom slide layout in Slide Master?
In Slide Master view, click "Insert Layout" to create a new custom layout. Then use the "Insert Placeholder" option to add content placeholders for text, pictures, charts, etc., as needed.
10 Marks How do you apply a different transition effect to each slide in a presentation?
Select each slide individually in the slide thumbnail pane, go to the Transitions tab, and choose a different transition effect for each selected slide.
10 Marks How do you insert and format a SmartArt graphic in PowerPoint?
Go to Insert > SmartArt, choose a category and layout, then add text in the text pane. Use SmartArt Design and Format tabs to change colors, styles, and formatting of the graphic elements.
10 Marks What is the difference between "Save As" and "Save a Copy" in PowerPoint?
"Save As" creates a new file and makes it the active presentation, while "Save a Copy" creates a duplicate file but keeps the original presentation open and active.
10 Marks How can you change the background of all slides at once?
Go to View > Slide Master, select the top master slide, right-click and choose Format Background, make your changes, and they will apply to all slides based on that master.
10 Marks How do you create a summary or agenda slide in PowerPoint?
In Slide Sorter view, select the slides you want to include in the summary, then go to Home > Slides > Section > Zoom Summary to create an interactive agenda slide.
10 Marks What is the "Selection Pane" and how does it help in managing objects?
The Selection Pane (Home > Select > Selection Pane) lists all objects on a slide, allowing you to select, rename, show, hide, or reorder them, which is especially useful when objects overlap.
10 Marks How can you create a photo album presentation in PowerPoint?
Go to Insert > Photo Album > New Photo Album, add pictures, adjust layout and formatting options, then click Create to generate a presentation with your selected photos.
10 Marks What are the different options for printing handouts in PowerPoint?
In Print settings, under Settings, choose "Handouts" and select how many slides per page (1, 2, 3, 4, 6, or 9), and whether to include slide frames, print comments, or arrange slides horizontally or vertically.

A. What is a slide?

A slide is a single page of a presentation created using software like Microsoft PowerPoint. A presentation is composed of a series of slides, which can contain text, images, charts, graphs, videos, and other multimedia elements to convey information to an audience.

B. Give an example of any four types of animations.

The four main types of animations in PowerPoint are:

1. Entrance: Animations that bring an object onto the slide. (e.g., Fade, Fly In, Zoom)

2. Emphasis: Animations that highlight or draw attention to an object already on the slide. (e.g., Spin, Pulse, Grow/Shrink)

3. Exit: Animations that make an object leave the slide. (e.g., Fade, Fly Out, Disappear)

4. Motion Paths: Animations that make an object move along a specified path. (e.g., Lines, Arcs, Loops, Custom Path)

C. Define SmartArt graphics.

SmartArt graphics are pre-designed visual representations of information in PowerPoint. They allow users to create professional-looking diagrams, such as organizational charts, process flows, cycles, or hierarchies, quickly and easily by converting bulleted lists into visual diagrams.

D. State different types of numbering in PowerPoints.

The different types of numbering in PowerPoint include:

1. Decimal Numbers: 1., 2., 3., ...

2. Roman Numerals: Uppercase (I., II., III., ...) or Lowercase (i., ii., iii., ...)

3. Alphabetical: Uppercase Letters (A., B., C., ...) or Lowercase Letters (a., b., c., ...)

4. Custom Numbering: Using different formats like 1), A), etc.

E. How to present a slide show?

To present a slideshow in PowerPoint:

1. Click on the "Slide Show" tab on the ribbon.

2. To start from the beginning, click "From Beginning" or press the F5 key.

3. To start from the current slide, click "From Current Slide" or press Shift + F5.

4. Use the Spacebar, Enter, or Right Arrow key to move to the next slide.

5. Press the Esc key to end the slideshow.

F. State about various views of PowerPoint.

The primary views in PowerPoint are:

1. Normal View: The default editing view, showing a slide pane, thumbnail pane, and notes pane.

2. Slide Sorter View: Displays all slides as thumbnails, ideal for rearranging, adding, or deleting slides.

3. Reading View: Presents the slideshow in a window, allowing you to see animations and transitions without going to full-screen mode.

4. Slide Show View: The full-screen presentation view used to deliver the presentation to an audience.

5. Notes Page View: Shows a page with the slide at the top and the speaker's notes below it.

G. How can you set custom timing for slide transition in PowerPoint presentation?

To set custom slide transition timing:

1. Select the slide(s) you want to apply the timing to.

2. Go to the "Transitions" tab.

3. In the "Timing" group, uncheck "On Mouse Click".

4. Check the "After" checkbox and enter the desired time (e.g., 00:05.00 for 5 seconds) in the box provided.

5. The slide will now advance automatically after the specified time.

H. Give an example of any two type of transition in PowerPoint presentation.

Two types of transitions are:

1. Subtle: Smooth and gentle effects. (e.g., Fade, Push, Wipe)

2. Dynamic: More dramatic and eye-catching effects. (e.g., Orbit, Fly Through, Vortex)

9. Write a short note on handouts.

Handouts are printed versions of a PowerPoint presentation that can be distributed to the audience. They help the audience follow along and take notes during the presentation and serve as a reference afterward.

Purpose: The main purpose is to provide a tangible takeaway for the audience, ensuring they remember the key points.

Layout Options: PowerPoint allows you to print handouts with 1, 2, 3, 4, 6, or 9 slides per page. The most common are 2 slides per page (saves paper) and 3 slides per page (provides lines for notes next to each slide).

Creation: To create handouts, go to File > Print. Under "Settings," click "Full Page Slides" and then select your desired layout from the "Handouts" section. You can also choose to print in Grayscale or Pure Black and White to save ink.

Header and Footer: You can add page numbers, the date, and custom text to the header or footer of the handout pages via the "Insert" tab.

10. Discuss about different bullets and numbering in PowerPoint.

Bullets and numbering are used to create lists in PowerPoint, making content more organized and readable.

Bullets: Used for unordered lists where the sequence of items is not important.

Types: Standard bullet points (•), arrows (→), checkmarks (✓), squares (■), and custom symbols or even images can be used as bullets.

Customization: You can change the bullet's symbol, color, and size by selecting the list, going to the "Home" tab, clicking the dropdown arrow next to the "Bullets" button, and selecting "Bullets and Numbering."

Numbering: Used for ordered lists where the sequence or steps are important.

Types: Decimal numbers (1., 2., 3.), uppercase/lowercase Roman numerals (I., II., i., ii.), and uppercase/lowercase letters (A., B., a., b.).

Customization: Similar to bullets, you can change the numbering format, start at a specific number, and change its color and size via the "Bullets and Numbering" dialog box.

Multilevel Lists: Both bullets and numbering can be indented to create sub-points or nested lists, providing a hierarchical structure to the information.

11. Write down the steps to add images in presentations.

To add images to a PowerPoint presentation:

1. Select the Slide: Navigate to the slide where you want to insert the image.

2. Insert Tab: Go to the "Insert" tab on the ribbon.

3. Pictures Option: In the "Images" group, click on "Pictures".

4. Choose Source:

· This Device: To insert an image stored on your computer. A dialog box will open; browse to the image file, select it, and click "Insert".

· Online Pictures: To insert an image from online sources like Bing Image Search or OneDrive.

5. Adjust the Image: Once inserted, you can click and drag the image to reposition it. Use the corner handles to resize it proportionally.

6. Format the Image (Optional): Select the image to activate the "Picture Format" tab. Here you can apply corrections, artistic filters, add borders (picture styles), crop the image, and adjust its brightness/contrast.

12. Discuss how to insert text in presentation and format it.

Inserting Text:

Text is primarily added using Text Boxes and Placeholders.

1. Placeholders: When you create a new slide (like a Title and Content slide), click inside the placeholder that says "Click to add text" and start typing.

2. Text Boxes: For freeform text anywhere on the slide, go to the "Insert" tab, click "Text Box", then click and drag on the slide to create a box, and type your text.

Formatting Text:

After selecting the text, use the tools in the "Home" tab:

· Font: Change the font type (e.g., Arial, Calibri), size, and color.

· Emphasis: Make text Bold, Italic, or Underlined.

· Alignment: Align text to the left, center, right, or justify it.

· Spacing: Adjust line spacing and character spacing.

· Bullets/Numbering: Add bullet points or numbering to lists.

· WordArt Styles: Apply pre-designed text effects from the "Format" tab under "Drawing Tools" for more dramatic styling.

13. How can we add tables in PowerPoint presentation?

To add a table in PowerPoint:

1. Select the Slide: Go to the slide where you want the table.

2. Insert Tab: Click on the "Insert" tab.

3. Table Option: Click on the "Table" button.

4. Choose Method:

· Grid Method: Hover over the grid to select the number of rows and columns (e.g., 4x5), then click to insert.

· Insert Table: Click "Insert Table..." and manually enter the number of columns and rows.

· Draw Table: Click "Draw Table" to draw the table boundaries and cell divisions manually with a pencil tool.

5. Enter Data: Click inside a cell and type to add your data.

6. Design the Table: Once the table is selected, the "Table Design" and "Layout" tabs appear. You can use these to apply table styles, change border colors, shade cells, and adjust cell alignment and size.

14. Explain about custom animation in PowerPoint and discuss about its various features.

Custom Animation is a powerful feature in PowerPoint that allows you to apply and control unique animation effects to individual elements on a slide (like text, images, shapes).

Steps to Apply Custom Animation:

1. Select the object you want to animate.

2. Go to the "Animations" tab.

3. Click on "Add Animation" and choose an effect from the Entrance, Emphasis, Exit, or Motion Path categories.

Key Features of Custom Animation (Managed in the Animation Pane):

· Effect Options: Allows you to specify the direction, shape, or sequence of the animation (e.g., fly in "From Bottom," spin "Clockwise").

· Timing:

· Start: Controls when the animation begins.

· On Click: Starts when you click the mouse.

· With Previous: Plays simultaneously with the previous animation.

· After Previous: Starts immediately after the previous animation finishes.

· Duration: How long the animation effect lasts (in seconds).

· Delay: Adds a pause before the animation starts.

· Animation Pane: A panel that appears on the side, listing all animations on the current slide in the order they play. It allows you to select, reorder, modify, and remove animations easily.

· Reordering: You can change the sequence of animations by dragging them up or down in the Animation Pane.

· Triggers: Advanced feature that starts an animation only when you click on a specific object on the slide.

15. Write a short note on slide master.

The Slide Master is the top slide in a hierarchy of slides that stores design information—themes, layouts, colors, fonts, backgrounds, and placeholder positions—for a presentation.

· Purpose: Its primary purpose is to ensure consistency across all slides. By making a change in the Slide Master (e.g., changing the font or adding a logo), that change is automatically applied to every slide that uses that layout, saving time and effort.

· Access: To view the Slide Master, go to the "View" tab and click "Slide Master".

· Components:

1. Slide Master: The top-most, larger thumbnail. Changes here affect all layout slides beneath it.

2. Layout Masters: The smaller thumbnails below the Slide Master. Each represents a different slide layout (e.g., Title Slide, Title and Content, Two Content). You can customize placeholders for each specific layout.

· Common Uses:

· Adding a company logo or watermark to every slide.

· Customizing the font style and color for all titles and content.

· Setting the default bullet style.

· Adjusting the size and position of text and content placeholders.

· Exiting: To return to normal view, click "Close Master View" on the Slide Master tab.

16. Write down the steps to create a PowerPoint presentation. Presentation should contain three slides. First slide contains the name of the topic. Second slide contains a chart and a textbox and the third slide contains description of the topic.

Steps to Create the Presentation:

1. Open PowerPoint: Launch Microsoft PowerPoint. A new blank presentation will open with a Title Slide layout.

2. Slide 1: Title Slide

· Click in the "Click to add title" placeholder and type the name of your topic, e.g., "The Future of Renewable Energy."

· Click in the "Click to add subtitle" placeholder and type your name or organization.

3. Add Slide 2: Chart and Textbox

· Go to the "Home" tab and click "New Slide". Choose the "Title and Content" layout.

· Click in the title placeholder and type "Current Energy Usage."

· To add a chart: In the content area, click the "Insert Chart" icon. Select a chart type (e.g., Column Chart), and click OK. A sample Excel sheet will open; edit the data with your own, then close the Excel window.

· To add a textbox: Go to the "Insert" tab, click "Text Box", then click below or next to the chart and type a caption or explanation, e.g., "Figure 1: Global energy consumption by source (2023)."

4. Add Slide 3: Description

· Go to the "Home" tab, click "New Slide", and again choose "Title and Content" or "Blank" layout.

· Add a title, e.g., "Detailed Description."

· Click in the content placeholder (or insert a text box) and type a detailed description of your topic. You can use bullet points for better readability.

5. Apply Design (Optional): Go to the "Design" tab and select a theme to apply a consistent color and font scheme to all slides.

6. Save the Presentation: Go to File > Save As, choose a location, give your presentation a name, and click "Save."

17. Briefly discuss about slide transition.

Slide transitions are visual effects that occur when you move from one slide to the next during a presentation. They add a professional touch and can make a presentation more dynamic and engaging.

· Purpose: The primary purpose is to create a visual break between slides, signaling to the audience a change of topic or idea. They should be used subtly to enhance the flow, not distract from the content.

· Applying Transitions:

1. Select the slide(s) you want to apply a transition to.

2. Go to the "Transitions" tab.

3. Browse the gallery of transition effects (e.g., Fade, Push, Wipe, Split, Uncover) and click on one to apply it.

· Customizing Transitions:

· Effect Options: Many transitions have options you can change, such as direction (e.g., Wipe From Left, Wipe From Top) or shape.

· Sound: You can add a sound effect to accompany the transition (use sparingly).

· Duration: This controls how fast the transition effect plays.

· Timing (Advancing Slides):

· On Mouse Click: The slide advances only when the mouse is clicked. This is the default and gives the presenter full control.

· After: The slide advances automatically after a set amount of time. This is used for self-running presentations (e.g., kiosks).

· Best Practice: It is often best to use a simple, consistent transition (like "Fade") throughout the entire presentation for a professional look, rather than using many different, flashy effects.
